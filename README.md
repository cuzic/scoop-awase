# scoop-awase

[awase](https://github.com/cuzic/awase)（親指シフト/NICOLA キーボードリマッパー）の Scoop バケットです。

このリポジトリのマニフェストは `awase` リポジトリの `release.yml` によって
新バージョンのタグ push のたびに自動更新されます。手動編集は不要です。

## インストール

```powershell
scoop bucket add awase https://github.com/cuzic/scoop-awase
scoop install awase
```

## アップデート

```powershell
scoop update awase
```
